번호판을 메인으로.

균질한 리스트에서 번호판이 차종 글자보다 작아 묻혔다. 번호판을 메인 정보로 승격하고, 정보 카드를 분리, 터치 영역을 S~XL로 표준화했다. 핸들을 드래그해 전·후를 비교해보세요.

SOLUTION 02 · 결제·정산

매출을 한 화면에서.

결제·정산이 ‘우리공장’ 탭 2단계 깊이에 묻혀 있었다. 결제현황을 메인 탭으로 승격하고 정산을 통합, 진입을 2단계에서 1단계로 줄였다.

결제내역이랑 정산현황을 한 화면에서 한 번에 보고 싶어요.

메인 탭 승격정산 통합진입 2→1단계
SOLUTION 03 · 작업사진

장갑 낀 손으로도,
한 번에.

사진 기능이 상단에 몰려 한 손으로 닿지 않았다. 엄지존 하단으로 내리고 진입점을 통합, 다중 선택과 일괄 공유를 더했다.

사진 여러 장 골라서 보내는 게, 장갑 낀 손으론 너무 번거로워요.

엄지존 배치진입점 통합다중 선택·일괄 공유
SOLUTION 05 · 현장 톡뷰 · RELEASED

누가 작업했는가.

계정이 하나라 ‘누가’를 식별할 수 없었다. 계정 분리는 공수로 보류하고, 작업자 이름 표시 + 공정 단계 그룹화로 핵심 가치부터 1차 릴리즈했다.

사진을 누가 찍었는지, 지금 누가 무슨 작업을 하는지 구분이 안 돼요.

작업자 이름 표시공정 단계 그룹진행 상황 공유
