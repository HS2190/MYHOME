# 안현서 — UI/UX Designer Portfolio

블랙 + 옐로우 에디토리얼 포트폴리오 웹사이트. *"From screens to systems."*

## 구조
- `index.html` — 메인 (브랜딩 페이지)
- `projects.html` — 프로젝트 목록
- `wepair.html` · `gocha.html` · `sensicon.html` — 케이스 스터디 상세
- `assets/site.css` · `assets/site.js` — 공유 스타일·모션
- `assets/` — 이미지 (폰 목업 스크린샷, 케이스 페이지 렌더)

## 배포
`main` 브랜치에 푸시하면 GitHub Actions(`.github/workflows/deploy.yml`)가 GitHub Pages로 자동 배포한다.

→ **https://hs2190.github.io/MYHOME/**
